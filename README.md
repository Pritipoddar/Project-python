# 🎨 Python Turtle Graphics & Interactive Games Gallery

A sleek, interactive web-based showcase of Python Turtle graphics and interactive games compiled directly in the browser using **Skulpt**. 

Explore algorithmic art generation and classic arcade gameplay built using Python!

---

## ✨ Features & Projects

### 🌸 1. Dynamic Flower Generator (`flower.py`)
* **Uniform Color Shifts:** Draws the entire flower structure in one unified color at a time.
* **Color Hierarchy:** Transitions smoothly through **Primary Shades** $\rightarrow$ **Secondary Shades** $\rightarrow$ finishes in **Baby Pink**.
* **Dynamic Palette:** Generates fresh randomized color shades on every run while strictly excluding green tones.

### 💖 2. Centered Animated Heart (`heart.py`)
* **Smooth Animation:** Features step-by-step drawing with a visible turtle cursor.
* **Centered Canvas Layout:** Scaled and positioned perfectly in the middle of a dark view container.

### 🐍 3. Interactive Snake Game (`snake.py`)
* **Real-time Gameplay:** Continuous game loop utilizing `ontimer()` for responsive browser performance.
* **Keyboard Controls:** Use `Arrow Keys` (Up, Down, Left, Right) to navigate and eat pink food.
* **Collision Detection:** Automatic detection for border wrapping and self-collision (skin-touching).
* **Game Over Screen:** Displays a stylized **GAME OVER** overlay alongside your final score upon collision.

### 🌲 4. Recursive Fractal Tree (`tree.py`)
* **Algorithmic Artwork:** Generates a fractal tree structure using recursive branch functions.
* **Vibrant Styling:** Brown trunk with orange leaf nodes centered on a dark canvas.
